## 🚀 Featured Project

<div align="center">

### 🗂️ Job Board Microservice Platform

</div>

> A production-style backend platform built with **Spring Boot**, **API Gateway**, **JWT Auth**, and **RabbitMQ** — designed to demonstrate real-world microservice architecture.

| Service | Description |
|---|---|
| 🔐 `user-service` | JWT authentication, Spring Security, BCrypt password hashing |
| 💼 `job-service` | Job CRUD, dynamic filtering with Specifications, Feign Client |
| 🌐 `api-gateway` | Centralized routing via Spring Cloud Gateway |
| 📨 `notification` | Async messaging via RabbitMQ |

**Stack:** `Spring Boot 4.x` · `Spring Cloud` · `PostgreSQL` · `RabbitMQ` · `Docker`
