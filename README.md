<br/>

> 🏗️ A **production-style backend platform** built with Spring Boot, Spring Cloud, JWT Auth & RabbitMQ — designed to demonstrate real-world microservice architecture patterns.

<br/>

| 🎨 | Service | Description |
|:---:|:---|:---|
| 🔐 | `user-service` | JWT authentication, Spring Security 6, BCrypt password hashing |
| 💼 | `job-service` | Job CRUD, dynamic filtering with JPA Specifications, Feign Client |
| 🌐 | `api-gateway` | Centralized routing via Spring Cloud Gateway |
| 📨 | `notification-service` | Async event-driven messaging via RabbitMQ |
| 📖 | `API Docs` | Springdoc OpenAPI 3.0 integrated across all services |
